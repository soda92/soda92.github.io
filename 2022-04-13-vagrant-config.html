Vagrant Configurations¶

vagrant init debian/bullseye

add proxy:

if Vagrant.has_plugin?("vagrant-proxyconf")
  config.proxy.http     = "http://192.168.56.1:7890"
  config.proxy.https    = "http://192.168.56.1:7890"
  config.proxy.no_proxy = "localhost,127.0.0.1"
end

install proxy plugin

vagrant plugin install vagrant-proxyconf

startup:

vagrant up

update images¶

vagrant box remove archlinux/archlinux

vagrant box add archlinux/archlinux

vagrant box remove ubuntu/focal64
vagrant box add ubuntu/focal64

    config.proxy.http     = "http://192.168.56.1:7890"
    config.proxy.https    = "http://192.168.56.1:7890"
